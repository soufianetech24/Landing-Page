Calm Landing Page
Mini projet en HTML + CSS pour recréer la landing page de Calm à partir d’une maquette.

Contenu du projet
index.html : structure de la page
style.css : styles et responsive
/images : images utilisées
README.md : description du projet
Objectif
Reproduire la maquette le plus fidèlement possible : couleurs, textes, marges, typographie et version mobile. Aucun JavaScript ni framework.

Sections incluses
Header, Hero, Services, Catalogue, Témoignages, Offre Premium, Image de transition, Footer.

Technologies
HTML, CSS
